begin dyndoc Using Stata dynamic tags in a text file with the dyndoc command =============================================================== ex.txt Let us consider an example where we study the mpg and weight variables in auto.dta. In our examples below, we will first write the commands so that they will be displayed in our target HTML file. Then, we will write the commands so that Stata will process the Stata dynamic tags, displaying the results of the Stata commands in the target HTML file. We first use the sysuse command to load the dataset and then describe the data using the describe command.

<<dd_do>>
sysuse auto, clear
describe
<</dd_do>>

This produces the following Stata results:
