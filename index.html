Welcome to my website!

The website is mainly used for my blog. I write in mostly Indonesian, but sometimes in english. Why in Indonesian? Well, I feel like we have already a lot of great Indonesian scholar these days, who writes a lot of good piece. However, mostly are written in english. Not that it is bad, but i have many reasons to write in indonesian. I believe in better access to education, and this is one way I can try to contribute to Indonesia! I do ocassionally write in english, both in this blog and other places.

Do check my about page for more information about me and my research, Scroll further down to see the content my blogs (find the "english" tag to find articles done in english), or just hit the blog tab! Thanks for stopping by!

Selamat datang di website saya!!

Nama saya Krisna, atau sering juga dipanggil Imed. Saya sekarang sedang menempuh pendidikan doktorat di bidang ekonomi di Australian National University, disponsori oleh Australia Awards. Thesis saya mengenai perdagangan internasional.

Website ini dibuat untuk mengakomodir kebutuhan personal website sebagai mahasiswa phd dan nantinya sebagai akademik. Banyak mahasiswa doktorat yang punya personal website yang isinya kayak CV aja gitu. Nah supaya situs saya gak sepi, sekalian aja saya jadiin blog. Sebelumnya saya sudah pernah ngeblog di medium dan wordpress. Meskipun Hosting di github lebih ribet daripada di medium dan wordpress, tapi bisa embed macem-macem (misalnya file pdf di CV dan youtube), dan bisa pake \(LaTeX\).

blog saya sendiri isinya macem-macem. lebih ke pengalaman dan pemikiran saya, serta hal-hal yang berbau kebijakan ekonomi, khususnya ekonomi makro, perdagangan internasional, dan pertumbuhan industri.

Di atas ada tautan untuk navigasi. Blog untuk semua page blog saya, tapi kalo mau nyari tags, skrol aja ke bawah. Cek juga tautan membuat blog dengan Jekyll dan github jika anda juga pengen coba dan pengen bantuin saya benerin web ini :D. Di postingan saya itu ada beberapa link sumber belajar saya. Namun di sini juga akan saya list beberapa yang sangat berperan membantu saya belajar bikin blog ini. Terima kasih pada oknum-oknum ini karena telah jadi sumber ilmu gratisan untuk saya.
